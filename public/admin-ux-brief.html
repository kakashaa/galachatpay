📋 ملخص تصميم لوحة تحكم الأدمن — Admin Dashboard UX Brief

🎯 الغرض

تصميم واجهة مستخدم (UI) احترافية للوحة تحكم إدارية لتطبيق غلا شات (Gala Chat) — تطبيق صوتي اجتماعي عربي.

📱 المنصة

  • نوع: Mobile-first Web App (عرض أقصى 430px)
  • اللغة: عربية (RTL)
  • التقنيات: React + TypeScript + Tailwind CSS + Framer Motion
  • الثيم: داكن فقط — خلفية #09090b

💡 ملاحظات للمصمم

  1. التصميم حصراً للموبايل (430px max)
  2. RTL — كل شي من اليمين لليسار
  3. Dark Mode فقط
  4. الأرقام بخط Monospace
  5. الـ Badges تظهر فقط إذا > 0
  6. الخدمات المخفية حسب الصلاحيات لا تظهر
  7. الشريط السفلي ثابت مع safe-area
  8. Press → Scale 90% → Release → Scale back
  9. Framer Motion spring animations
  10. مؤقت الدوام يتغير لونه تلقائياً
