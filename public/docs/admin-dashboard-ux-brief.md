# 📋 ملخص تصميم لوحة تحكم الأدمن — Admin Dashboard UX Brief

## 🎯 الغرض من هذا الملف
هذا الملف يحتوي على كل التفاصيل اللازمة لتصميم واجهة مستخدم (UI) احترافية للوحة تحكم إدارية لتطبيق **غلا شات** (Gala Chat) — تطبيق صوتي اجتماعي عربي.

---

## 📱 المنصة والتقنيات
- **نوع التطبيق:** Mobile-first Web App (عرض أقصى 430px)
- **اللغة:** عربية (RTL — من اليمين لليسار)
- **التقنيات:** React + TypeScript + Tailwind CSS + Framer Motion
- **الخطوط:** Tajawal (عناوين) + IBM Plex Sans Arabic (نصوص) + Mono للأرقام
- **الثيم:** داكن (Dark Mode فقط) — خلفية `#09090b`

---

## 👥 أنواع المستخدمين الإداريين (الأدوار)

| الدور | الصلاحيات |
|-------|-----------|
| **Owner** (المالك) | كل الصلاحيات + إدارة الحسابات + السجل + الإعدادات |
| **Super Admin** | إدارة VIP + الحماية + الرواتب + الطلبات + الدعم |
| **Admin** (مشرف) | صلاحيات محدودة حسب ما يحدده المالك |
| **Moderator** (مراقب) | صلاحيات مراقبة فقط |

## 🏠 الصفحة الرئيسية (Home Dashboard)

### البنية من الأعلى للأسفل:

### 1️⃣ الهيدر (Header)
- **يسار:** نص "لوحة التحكم" + اسم الأدمن "أهلاً، [الاسم]"
- **يمين:** زر الإشعارات (جرس) مع Badge للعدد

### 2️⃣ شريط البحث (UUID Search)
- حقل إدخال بتنسيق `font-mono` يقبل UUID (معرف رقمي)
- زر بحث على اليسار
- عند البحث يختفي كل محتوى الداشبورد ويظهر **بطاقة تحكم المستخدم**

### 3️⃣ بطاقة تحكم المستخدم (User Control Panel) — تظهر عند البحث
عند إدخال UUID والبحث، تظهر واجهة كاملة:

#### أ. معلومات المستخدم:
- **الصورة:** دائرية 80px مع حلقة حدود
- **الاسم:** بخط عريض
- **UUID:** بخط mono تحت الاسم
- **حالة الاتصال:** شارة (Online = أخضر / Offline = رمادي)
- **شارة VIP:** مستوى VIP (إذا موجود) — بلون ذهبي
- **تاريخ الانضمام**

#### ب. شبكة إحصائيات (4 أعمدة):
| الإحصائية | اللون |
|-----------|-------|
| الراتب ($) | أخضر |
| مستوى الداعم | أزرق |
| مستوى الدعم | بنفسجي |
| مستوى الشحن | سماوي |

#### ج. معلومات تفصيلية:
- صافي الراتب
- الخصومات (أحمر)
- رقم الوكالة
- رقم العائلة
- الحالة (نشط / محظور)

#### د. أزرار التحكم السريع (شبكة 3 أعمدة × 2 صفوف):
| الزر | الأيقونة | اللون | الوظيفة |
|------|---------|-------|---------|
| إرسال VIP | Star ⭐ | ذهبي | ينتقل لصفحة VIP مع UUID |
| حظر / فك حظر | ShieldBan / Unlock | أحمر / أخضر | حظر أو فك حظر المستخدم |
| تغيير آيدي | Hash # | بنفسجي | ينتقل لصفحة تغيير ID |
| تصفير الراتب | DollarSign $ | برتقالي | يصفّر الراتب |
| إيقاف الشحن | XCircle ❌ | أحمر | يوقف الشحن |
| تغيير الصورة | Camera 📷 | سماوي | يغيّر صورة المستخدم |

### 4️⃣ كروت الإحصائيات (Stats Cards) — تظهر في الصفحة الرئيسية
شبكة 3 أعمدة:

| الإحصائية | اللون | الوصف |
|-----------|-------|-------|
| معلقة | ذهبي/أصفر | طلبات بانتظار المراجعة |
| مقبولة | أخضر | طلبات تمت الموافقة عليها |
| مرفوضة | أحمر | طلبات تم رفضها |

### 5️⃣ مؤقت الدوام (Shift Timer)
- يعرض وقت بداية ونهاية الدوام
- شريط تقدم متحرك (Progress Bar)
- الألوان: أخضر → أصفر (75%+) → أحمر (انتهى)
- تنبيه صوتي قبل 5 دقائق من نهاية الدوام

### 6️⃣ أزرار الدردشة السريعة
صفان أفقيان:
- **المشرفين** — أيقونة درع + لون أخضر
- **كل الأدمن** — أيقونة مستخدمين + لون رمادي

### 7️⃣ شبكة الخدمات (Service Grid) — 4 أعمدة

| # | الخدمة | الأيقونة | اللون | المسار | Badge؟ |
|---|--------|---------|-------|--------|--------|
| 1 | VIP | Star | ذهبي | /admin/vip | ✅ عدد الطلبات المعلقة |
| 2 | الحماية | ShieldBan | وردي/أحمر | /admin/ban | ✅ عدد البلاغات |
| 3 | الرواتب | DollarSign | أخضر | /admin/salary | ✅ عدد طلبات السحب |
| 4 | الطلبات | ClipboardList | سماوي | /admin/gifts | ✅ عدد الطلبات |
| 5 | المتجر | Gift | وردي | /admin/gifts | ❌ |
| 6 | الدعم | Headset | سماوي | /admin/support | ✅ عدد التذاكر |
| 7 | آيدي | Hash | بنفسجي | /admin/id-change | ❌ |
| 8 | المداخيل | Wallet | أخضر زمردي | /admin/income | ❌ |
| 9 | الوكالات | Briefcase | برتقالي | /admin/agencies | ❌ (Owner فقط) |
| 10 | الأدمن | Users | تيل | /admin/accounts | ❌ (Owner فقط) |
| 11 | السجل | ScrollText | نيلي | /admin/log | ❌ (Owner فقط) |
| 12 | الإعدادات | Settings | رمادي | /admin/settings | ❌ (Owner فقط) |

**ملاحظة:** الخدمات 1-4 تظهر فقط لـ Super Admin وأعلى. الخدمات 9-12 تظهر فقط للـ Owner.

### 8️⃣ آخر العمليات (Recent Activity Log)
- قائمة عمودية بآخر 4 عمليات
- كل عملية تعرض: نقطة ملونة (أخضر للموافقة / أحمر للرفض) + نص العملية + الوقت
- زر "عرض الكل" ينقل لصفحة السجل (للـ Owner فقط)

## 🧭 شريط التنقل السفلي (Bottom Navigation Dock)

شريط عائم (Floating Dock) بتأثير زجاجي (Glass-morphism):

| # | التبويب | الأيقونة | الوظيفة |
|---|--------|---------|---------|
| 1 | الرئيسية | Home | الصفحة الرئيسية |
| 2 | بحث | Search | بحث عام |
| 3 | **الدردشة** | MessageCircle | **زر مركزي كبير أخضر بارز** — ينقل لصفحة الدردشة |
| 4 | مراقبة | Eye | صفحة المراقبة |
| 5 | المزيد | MoreHorizontal | قائمة إضافية |

**خصائص الزر المركزي (الدردشة):**
- أكبر من باقي الأزرار (50×50px)
- لون أخضر زمردي (emerald)
- ظل متوهج أخضر
- يعرض Badge عدد الرسائل غير المقروءة

**خصائص التبويب النشط:**
- الأيقونة والنص بلون أخضر زمردي
- (اختياري) نقطة مؤشر متحركة أسفل الأيقونة النشطة

## 📐 مواصفات التصميم

### الأبعاد:
- **عرض أقصى:** 430px (mobile-first)
- **Padding أفقي:** 16px (px-4)
- **المسافات العمودية بين الأقسام:** 24px (space-y-6)
- **حجم أيقونات الخدمات:** 52×52px مع border-radius 14px
- **حجم أيقونات الـ Dock:** 20×20px
- **زر الدردشة المركزي:** 50×50px مع border-radius 16px

### الخطوط:
- **عناوين رئيسية:** 18px bold
- **عناوين ثانوية:** 11px
- **أرقام الإحصائيات:** 24px mono bold
- **تسميات الخدمات:** 10px
- **Badges:** 8px bold

### الحواف (Border Radius):
- كروت كبيرة: `rounded-2xl` (16px)
- كروت صغيرة: `rounded-xl` (12px)
- أيقونات: `rounded-[14px]`
- الـ Dock: `rounded-[20px]`
- الأزرار: `rounded-xl` to `rounded-2xl`

### التأثيرات:
- **Glass-morphism:** `backdrop-blur-xl` + خلفية شفافة
- **Badges:** كرات حمراء صغيرة (16px) في أعلى يمين الأيقونة
- **Active state:** `scale(0.9)` عند الضغط
- **Hover:** لا يوجد (mobile-first)
- **الظلال:** ظل أخضر متوهج على زر الدردشة فقط

## 💡 ملاحظات للمصمم

1. **التصميم حصراً للموبايل** — لا حاجة لتصميم Desktop
2. **RTL (من اليمين لليسار)** — كل النصوص والأيقونات بالعربي
3. **Dark Mode فقط** — لا يوجد وضع فاتح
4. **الأرقام تكون بخط Mono** — للتمييز البصري
5. **الـ Badges** تظهر فقط إذا كانت القيمة > 0
6. **الخدمات المخفية** حسب صلاحيات الدور (لا تظهر أصلاً، لا تكون رمادية)
7. **شريط التنقل السفلي ثابت دائماً** — مع مراعاة `safe-area-inset-bottom` لأجهزة iPhone
8. **التفاعلات:** Press → Scale down 90% → Release → Scale back
9. **الانتقالات:** Framer Motion مع `spring` type للحركات الطبيعية
10. **مؤقت الدوام** يتغير لونه تلقائياً حسب الوقت المتبقي
